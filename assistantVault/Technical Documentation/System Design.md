# System Design

This note describes the base structure of the Personal Communication & Action Hub (unified inbox, analytics, and task management).

## Core components
- **Ingestion**: Collects messages and calls from multiple channels.
- **Core**: Persistent storage of timeline and metadata.
- **AI Analysis**: Generates concise summaries, suggested replies, and tasks.
- **Obsidian**: Primary interface and long-term memory.
- **Tasks**: Note-based task system.
- **Telegram Reminders**: Smart reminders via bot.
