# Product Specification

This specification describes the core functionality of the Personal Communication & Action Hub.

## Features
- Integration with Telegram, WhatsApp, Outlook, and Teams.
- Unified event timeline.
- Message analysis with AI.
- Task management and reminders via Telegram.

## Hardware requirements
- Web browser support.
- Local Obsidian storage.
